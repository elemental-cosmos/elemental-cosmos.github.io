# The Elemental Cosmos 🌌

A novel arrangement of the chemical elements — organized by **proximity to human life** rather than atomic number.

## Live Demo

Visit: `https://YOUR-USERNAME.github.io/elemental-cosmos/`

## How It Works

- **Hover** any element to see its details
- **Click** orbit rings to filter by layer
- **Click** legend colors to filter by functional family

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
